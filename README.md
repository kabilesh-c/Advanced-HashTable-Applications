# Real World HashMap Systems

This repository contains **10 real-world system design problems implemented using Hash Tables and HashMaps in Java**.

These problems simulate high-scale systems used by companies like Google, Amazon, Netflix, and Twitter.

## Problems Included

1. Social Media Username Availability Checker
2. E-commerce Flash Sale Inventory Manager
3. DNS Cache with TTL
4. Plagiarism Detection System
5. Real-Time Website Analytics Dashboard
6. Distributed API Rate Limiter
7. Search Engine Autocomplete System
8. Smart Parking Lot (Open Addressing)
9. Financial Transaction Fraud Detection (Two-Sum Variants)
10. Multi-Level Cache System (L1/L2/L3)

## Key Concepts

- Hash Tables
- O(1) Lookup Systems
- Collision Resolution
- Frequency Counting
- Load Factor Management
- Real-time Analytics
- Distributed Systems

- Java
- HashMap

This repository is created for **placement preparation and system design practice**, focusing on how hash tables power real-world high-performance systems.
